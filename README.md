# HR System Server

## Description

The Backend server for HR System interface.

## Tech Stack

- [Nestjs](https://nestjs.com/), [Mongoose](https://mongoosejs.com/).

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
